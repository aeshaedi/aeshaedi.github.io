All about Aeshaedi: Aeshadi is my name. My name is inspired by the persian melodic language. My name is used to present myself to many people. There are a lot of people in the Earth though, so I made this website. For everyone to be able to know Aeshaedi.

The nature of my name

My name is sometimes written with the letter "Æ". Like this: Æshædi. This is for aesthetic reasons. Although it can make also a good phonetic transcription.

The phonetic transcription of my name would be [ae'sha'edi]
